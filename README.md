# Unity Shooting Game

## 🎮 概要

Unityを使用して制作した2Dシューティングゲームです。
プレイヤーが敵を倒しながら進むシンプルなゲームを作成しました。

## 🕹 操作方法

* 移動：矢印キー or WASD
* 攻撃：スペースキー

## ⚙ 使用技術

* Unity（バージョン：〇〇）
* C#
* Visual Studio

## ✨ 工夫した点

* 敵の出現パターンを調整し、ゲームの難易度をバランス調整した
* 敵がプレイヤーの位置を狙って弾を撃つように実装
* スコア表示機能を追加し、ゲーム性を向上させた

## 🔧 今後の改善点

* ボス敵の追加
* UIの改善
* SEやBGMの追加
